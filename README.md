# odin-recipes
A recipe created in html for the Odin project

Introduction:

This project was created using Visual Studio Code to practice structuring, writing, and pushing the finished code to GitHub. It also allowed me to practice making clean code while building a simple recipe webpage. The goal of the project was to apply fundamental web development concepts by organizing a recipe into a properly structured document using HTML and supporting files.

While developing this project, I practiced writing and editing code in VS Code, structuring content with semantic HTML elements, and organizing project files in a clear and logical way. The recipe served as the content for the page, allowing me to focus on formatting information such as ingredients, instructions, and headings in a readable and well-structured layout.

I created an index.html page for odin-recipes.
I created three Recipe pages: broccoliCheddarSoup.html, chickenPotPieSoup.html, and lasagna.html in the recipe folder.
I created three images in the image folder: broccoliCheddarSoup.jpg, chickenPotPieSoup.jpg, and lasagna.jpg.

On each recipe page: 
	-added a title using h1
	-added a link to my image
	-used the following elements:
		<!-- comment --> : to add comments to my code
		-<html></html> : Open and closing html identifiers
		-<meta></meta> : Used to add metadat
		-<title></title> : Used to add a title to the sheet
		-<link> : used to add a links
		-<header></header> : identify the header data (what works in background (don't see on web page))
		-<body></body> : identify the main body of the data (what you see on web page)
		-<section></section> : creates different sections in the body
		-<ul></ul> : creates unordered list
		-<ol></ol> : creates ordered list
		-<li></li> : list items
		-<a></a> : anchor point 
		-<p></p> : general text
		-<i></i> : italicize text
		-<strong></strong> : makes text bold and marks as important
		-<footer></footer> : copyright information, address information, or notes for outside source data.
