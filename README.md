# odin-recipes
A recipe created in html for the Odin project

Introduction:

This project was created using Visual Studio Code to practice structuring, writing, and pushing the finished code to GitHub. It also allowed me to practice making clean code while building a simple recipe webpage. The goal of the project was to apply fundamental web development concepts by organizing a recipe into a properly structured document using HTML and supporting files.

While developing this project, I practiced writing and editing code in VS Code, structuring content with semantic HTML elements, and organizing project files in a clear and logical way. The recipe served as the content for the page, allowing me to focus on formatting information such as ingredients, instructions, and headings in a readable and well-structured layout.
